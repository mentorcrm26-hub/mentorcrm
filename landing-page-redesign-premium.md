# Task: Landing Page Redesign - Premium Stealth Precision

## 1. Analysis
- **Current State**: Standard SaaS landing page with centered hero, purple/indigo palette, 3-column grid features. Generic copy and standard rounding.
- **Goal**: Transform into a premium, world-class landing page for financial agents.
- **Strategy**: 
    - Apply "Stealth Precision" design style.
    - Focus on Asymmetric layout.
    - Replace Purple with Electric Emerald/Zinc.
    - Implement sharp geometry.
    - Add scroll-triggered animations and micro-interactions.

## 2. Planning
- **Phase 1: Foundation (CSS & Layout)**
    - Redesign `src/app/page.tsx` structure.
    - Add custom grid and spacing.
- **Phase 2: Navigation**
    - Create a minimal, sharp navigation bar with glassmorphism (solid borders).
- **Phase 3: Hero Section (The "Wow" Factor)**
    - Massive typographic headline.
    - Overlapping dashboard preview (CSS-based or optimized image).
- **Phase 4: Features Section**
    - Fragmented grid layout.
    - Staggered entrance animations.
- **Phase 5: Pricing & CTA**
    - Premium card design with sharp edges and subtle grain/depth.
- **Phase 6: Polish**
    - Add `framer-motion` for fluid animations.
    - Final UX audit.
